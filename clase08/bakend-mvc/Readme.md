
# Flujo Fina del backend

1. `src/app.js` iniciar Express
2. `src/config/config.js` cargar variables de entorno
3. `src/config/db.js` conectar MongoDB
4. `src/routes/productosRoutes.js` define los endpoints
5. `src/controllers/productoController.js` ejecutar la logica de negocio
6. `src/models/Producto.js` define el esquema de negocio
7. Frontend / Las vistas Handlebars renderizan formularios y listados
8. Captura de Errores
9. Mostrar los errores
