# Tareas para la próxima clase

Resolver las siguientes actividades utilizando **Node.js**.  
Todos los ejercicios deben poder ejecutarse desde la terminal con:

node nombre_del_archivo.js

-----------------------------------------------------------------------

# Tarea 1 — Primer script en Node

Crear un archivo llamado:

tarea1.js

Declarar las siguientes variables:

- nombre
- edad
- ciudad

Mostrar un mensaje en consola utilizando **template strings** que incluya esos tres datos.

Ejecutar el archivo con Node desde la terminal.

-----------------------------------------------------------------------

# Tarea 2 — Función de cálculo

Crear un archivo llamado:

tarea2.js

Crear una función llamada:

calcularAreaRectangulo

La función debe recibir:

- base
- altura

La función debe **calcular el área del rectángulo** y mostrar el resultado en consola.

Luego llamar a la función con valores de prueba.

-----------------------------------------------------------------------

# Tarea 3 — Uso de objetos

Crear un archivo llamado:

tarea3.js

Crear un objeto llamado:

producto

El objeto debe tener las siguientes propiedades:

- nombre
- precio
- stock

Mostrar en consola la información del producto utilizando **template strings**.

-----------------------------------------------------------------------

# Tarea 4 — Crear una clase

Crear un archivo llamado:

tarea4.js

Crear una clase llamada:

Usuario

La clase debe tener:

- propiedad `nombre`
- propiedad `email`

Debe incluir un método llamado:

mostrarInfo()

Este método debe mostrar en consola la información del usuario.

Luego crear una instancia de la clase y ejecutar el método.

-----------------------------------------------------------------------

# Tarea 5 — Manejo de errores con try/catch

Crear un archivo llamado:

tarea5.js

Crear una función llamada:

dividir

La función debe recibir dos números.

Si el segundo número es **0**, la función debe lanzar un error.

Luego usar **try/catch** para ejecutar la función y manejar el error mostrando un mensaje en consola.

-----------------------------------------------------------------------

Para la próxima clase deben entregar una carpeta con los archivos:

tarea1.js  
tarea2.js  
tarea3.js  
tarea4.js  
tarea5.js  

Todos los archivos deben ejecutarse correctamente con Node desde la terminal.
